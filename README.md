# 💰 Expense Tracker

A modern expense tracker built using HTML, CSS, and Vanilla JavaScript.

## ✨ Features

- ➕ Add transactions
- ✏️ Edit transactions
- 🗑️ Delete transactions
- 💾 Local Storage
- 🔍 Search transactions
- 🎯 Filter by type
- 📅 Sort by date
- 📤 Export CSV
- 🌙 Dark Mode
- 📱 Responsive Design

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
